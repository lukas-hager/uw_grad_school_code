## 3 by 3 Game Practice

### Question 1

    ##       a     b    c
    ## 1:  0,0   3,2 -2,2
    ## 2: -3,2  -3,0  2,3
    ## 3:  2,0 -1,-3  3,3

Proposed Solution:

  - Pure Strategies: (1;b), (3;c)
  - Mixed Strategies: (3/5, 0, 2/5; 0, 5/9, 4/9)
  - Prediction
      - (3;c) WD (1;b)
      - (3;c) PD (1;b), (3/5, 0, 2/5; 0, 5/9, 4/9)
      - (3;c) RD (1;b)
      - (3;c) SE
      - (3;c) CPNE
